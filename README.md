  This app enables user to add expenses and deposits to the their budget with/without online connection.

  ## [Installation](#table-of-contents)
  Clone the repository and open in VS Code.
  Run:
  * npm i install
  * npm i express
  * npm i mongoose
  Lastly:
  * npm start

  ## [Usage](#table-of-contents)
